# FARS Logo Specification

## Design Concept
Combine Filecoin (storage) + AI agent (intelligence) + reputation (trust)

## Elements
- Filecoin "F" icon
- Network/graph nodes (representing agents)
- Shield or badge (reputation/trust)

## Color Scheme
- Primary: #0090FF (Filecoin blue)
- Secondary: #00D395 (trust/verification green)
- Accent: #1A1B1F (dark background)

## Formats Needed
- logo.svg (vector, scalable)
- logo-512.png (for favicon/social)
- hero-1200x630.png (for submission/OG image)

## Style
- Modern, clean
- Geometric shapes
- Minimal gradients
- High contrast for readability
